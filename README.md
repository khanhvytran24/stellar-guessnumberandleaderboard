TÊN DỰ ÁN:
🎮🏆 Stellar Guess Game with Leaderboard

VẤN ĐỀ (1 câu):
Người dùng thiếu các ứng dụng Web3 có tính tương tác cao và lưu trữ dữ liệu minh bạch để hiểu rõ cách hoạt động của smart contract.

GIẢI PHÁP (1 câu):
dApp này sử dụng smart contract trên Stellar để xây dựng một trò chơi đoán số kết hợp hệ thống lưu điểm và bảng xếp hạng hoàn toàn on-chain.

TÍNH NĂNG STELLAR SỬ DỤNG:
[ ] Chuyển XLM/USDC  [ ] Token tùy chỉnh  [x] Soroban contract
[ ] DEX tích hợp    [ ] Trustline     [ ] Clawback/Tuân thủ

NGƯỜI DÙNG MỤC TIÊU:
Sinh viên IT, developer Web3, và những người muốn tìm hiểu cách xây dựng dApp có logic + lưu trữ dữ liệu trên blockchain.

TÍNH NĂNG CỐT LÕI (MVP):
Người dùng gọi hàm guess(number) để đoán số, nếu đoán đúng sẽ tự động được cộng điểm và lưu vào leaderboard on-chain.

TẠI SAO STELLAR:
Stellar cung cấp phí giao dịch cực thấp và thời gian xử lý nhanh (vài giây), giúp các ứng dụng tương tác như game hoạt động mượt mà, trong khi hệ thống truyền thống cần backend server và dễ bị thay đổi dữ liệu.
